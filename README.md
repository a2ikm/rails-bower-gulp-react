rails-bower-gulp-react
======================

react.jsとかそこら辺の勉強用に作ったプロジェクト。


## セットアップ

    $ bin/setup

この中でgemのインストールなどとあわせて、package.jsonからbowerをインストール、さらにそれをフックしてbower.jsonからreact等をインストールしている。
